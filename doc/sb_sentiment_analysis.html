This is the module responsible of the sentiments analysis.

Authors: Benjamin Schubert, Basile Vu, Ioannis Noukakis and Sarra Berich.

Description

This is the module responsible of the sentiments analysis. It manages the storage as well as the sentiment regex evaluations.
